Calling options¶

Command line

Description

ceda-cc -p <project> -D <directory> [..]

check all data in directory tree under specified directory

ceda-cc -p <project> -d <directory> [..]

check all data in specified directory

ceda-cc -p <project> -f <file> [..]

check a single file

ceda-cc –sum <log file directory> [..]

create an overview of results

ceda-cc –copy-config <target diectory> [..]

copy configuration files to a new directory

c4.main_entry()[source]¶

Wrapper around main() for use with setuptools. This function will intercept arguments –sum, –unitTest, -v, otherwise pass the commandline to c4_run.main.
